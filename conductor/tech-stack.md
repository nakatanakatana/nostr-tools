# Technology Stack

## Programming Language
- **Go**: Version 1.25.5 or later.

## Core Libraries
- **Nostr Protocol**: `github.com/nbd-wtf/go-nostr` - For handling Nostr primitives and protocol logic.
- **Relay Framework**: `github.com/fiatjaf/relayer` - For building the Nostr relay implementation.

## Utilities
- **Configuration**: `github.com/caarlos0/env` - For parsing environment variables into Go structures.
- **Logging**: `log/slog` - Standard library structured logging.

## Deployment
- **Containerization**: Docker (for ease of hosting and consistent environments).
